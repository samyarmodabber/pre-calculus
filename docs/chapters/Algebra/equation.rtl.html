4  معادلات جبری

4.1 معادله درجه اول

4.2 معادله

اگر در یک تساوی یک یا چند حروف وجود داشته باشد و آن تساوی به ازای بعضی از مقادیر آن حرف یا حروف درست باشد آنرا معادله گویند.

4.3 مثال

\[x - 2 = 0\]

\[2x^{2} - 3x - 5 = 0\]

4.4 مجموعه جواب معادله

مجموعه مقادیری از دامنه متغیر را به ازای آنها معادله به یک تساوی درست تبدیل می شود را مجموعه معادله نامند.

متغیر : متغیر حرف یا نمادی است که برای نامیدن عضو های یک مجموعه معینی به نام دامنه متغیر بکار می رود.

هر معادله که پس از ساده شدن به صورت \(ax + b = 0\) باشد را یک معادله درجه اول نامند.

4.5 حل کردن معادله

برای حل کردن معادله از دو خاصیت زیر استفاده می کنیم

  1. به طرفین معادله می توان یک مقدار اضافه و یا از طرفین آن یک مقدار کم کرد ،به عبارت دیگر در یک معادله می توان یک جمله را با عوض کردن علامت آن به طرف دیگر تساوی منتقل کرد.

(جمله مساوی در دو طرف معادله را می توان حذف کرد ).

  1. طرفین معادله را می توان در یک عدد غیر صفر ضرب و یا بر عددی غیر صفر تقسیم کرد.

حل معادله درجه اول \(:ax + b = 0\)

\[ax + b = 0 \Rightarrow ax = - b\]

اگر \(a \neq 0\ آنگاه\ x = - \frac{b}{a}\) که آن را جواب معادله گویند. اگر \(a = 0\) آنگاه معادله به صورت \(x = - b\) در می آید که :

الف : اگر \(b = 0\) آنگاه معادله به صورت \(x = 0\) در می آید که بیشمار جواب دارد.

ب : اگر \(b \neq 0\) آنگاه معادله جواب ندارد.

4.6 مثال

معادله \(5x + 2 = 2x + 11\) را حل کنید.

\[5x - 2x = 11 - 2 \Rightarrow 3x = 9 \Rightarrow x = \frac{9}{3} = 3\]

4.7 مثال

معادله : \(2(x - 1)^{2} - 3(x + 2)^{2} = - x(x + 8) + 6\) را حل کنید.

\[2\left( x^{2} - 2x + 1 \right) - 3\left( x^{2} + 4x + 4 \right) = - x^{2} - 8x + 6\]

\[2x^{2} - 4x + 2 - 3x^{2} - 12x - 12 = - x^{2} - 8x + 6\]

\[2x^{2} - 4x - 3x^{2} - 12x + 8x + x^{2} = 12 - 2 + 6\]

\[- 8x = 16 \Rightarrow x = \frac{16}{- 8} \Rightarrow x = - 2\]

4.8 مثال

معادله \(\frac{x - 1}{2} + \frac{x + 2}{6} = \frac{x + 1}{3}\) را حل کنید.

حل : طرفین معادله را در کوچکترین مضرب مشترک بین مخرجها ضرب می کنیم لذا داریم :

\[\left( \frac{x - 1}{2} + \frac{x + 2}{6} = \frac{x + 1}{3} \right).6 \Rightarrow 3(x - 1) + x + 2 = 2(x + 1) \Rightarrow 3x - 3 + x + 2x = 2 \Rightarrow 4x - 2x = 2 - 2 + 3 \Rightarrow 2x = 3 \rightarrow x = \frac{3}{2}\]

4.9 مثال

معادله \(ax + 4 = 2x + a^{2}\) را در نظر بگیرید.به ازای چه مقدار از معادله مبهم ؟ (بیشمار جواب دارد ).

حل :

\[ax - 2x = a^{2} - 4\ \]

\[x(a - 2) = a^{2} - 4\ \ \]

\[\left\{ \begin{matrix} a - 2 = 0 \\ a^{2} - 4 = 0 \\ \end{matrix} \right.\ \ \ \ \Rightarrow a = 2\]

\[است\ مبهم\ معادله\]

4.10 مثال:

معادله \(m^{2}x - 1 = x + m\) به ازای چه مقدار از \(x\) معادله مبهم و به ازای چه مقدار از m معادله جواب ندارد؟

حل :

\[m^{2}x - x = x + 1\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \Rightarrow x\left( m^{2} - 1 \right) = m + 1\]

\[\left\{ \begin{matrix} m^{2} - 1 = 0 \\ m + 1 = 0\ \\ \end{matrix} \right.\ \ \ \Rightarrow \ \ \ \ \ \ m = - 1\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ مبهم\ \ \ معادله\]

\[\left\{ \begin{matrix} m^{2} - 1 = 0 \\ m + 1 \neq 0\ \\ \end{matrix} \right.\ \ \ \ \Rightarrow \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ m = 1\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ ندارد\ جواب\ معادله\]

4.11 تمرین:

معادلات زیر را حل کنید.

  1. \(\frac{(x - 1)^{2}}{3} + \frac{(x + 3)^{2}}{6} = \frac{(x - 1)(x - 2)}{2}\ \)

  2. \(3(x + 4) - 5(x - 2) = 4(3x - 1) + 82\ \ \)

  3. \(\frac{7x + 9}{4} + x = \frac{5x}{2} + 3\ \ \)

  4. \(\frac{4x + 3}{5} - \frac{2x - 3}{3} = \frac{x - 1}{6}\ \)

  5. \(\frac{x + 2}{x - 2} - \frac{1}{x} = \frac{2}{x^{2} - 2x}\ \ \)

  6. \(\ \frac{x^{2}}{x - 1}\frac{x^{2}}{x + 1} = \frac{2x}{x^{2} - 1}\ \ \ \)

  7. \(\frac{12}{x - 7} - \frac{5}{x - 1} = \frac{7}{x - 10}\)

4.12 معادله های درجه دوم

4.13 تعریف:

هر معادله یک مجهولی به صورت \(ax^{2} + bx + c = 0\) را که در ان \(a,b,c\) اعداد حقیقی هستند و \(a \neq 0\)

یک معادله یک مجهولی در جه دوم می نامیم اعداد \(c,b,a\) را ضریب های معادله می گوییم.

4.14 مثال:

هر یک از معادله های \(3x^{2} + x = 0\ ,\ x^{2} - 7x + 12 = 0\) یک معادله در جه دوم هستند. منظور از حل یک معادله در جه دوم تعیین مجموعه جواب آن است.

روش های حل معادله درجه دوم :

الف ) حل معادله دوم با استفاده از تجزیه :

در صورتی که بتوان معادله درجه دوم را به عامل های اول تجزیه کرد می توان آن را به کمک معادله درجه اول حل کرد.

4.15 مثال:

معادلات زیر را حل کنید:

\[1) - 2x^{2} + 3x = 0\ \ \ \]

\[- 2x^{2} + 3x = x( - 2x + 3) = 0\ \]

حال از این ویژگی اعداد حقیقی استفاده می کنیم که :

((اگر حاصل ضرب دو عدد صفر باشد حد اقل یکی از آن ها صفر است )).

به بیان دیگر ، اگر \(xy = 0\) آنگاه x=0 یا y=0 بنابراین از \(x( - 2x + 3) = 0\) نتیجه می شود که \(x = 0\) یا \(- 2x + 3 = 0\) در نتیجه \(x = 0\) یا \(x = \frac{3}{2}\)

\[2)\ 4x^{2} - 9 = 0\]

برای حل این معادله از اتحاد مزدوج استفاده می کنیم :

\[(a + b)(a - b) = a^{2} - b^{2}\]

\[ax^{2} - 9 = (2x - 3)(2x + 3) = 0 \Rightarrow 2x - 3 = 0\ \ یا\ 2x + 3 = 0 \Rightarrow x = \frac{- 3}{2}\ \ یا\ x = \frac{3}{2}\]

\[3)x^{2} - 6x + 8 = 0\]

برای حل اینگونه معادله ها از اتحاد یک جمله ای مشترک استفاده می کنیم :

\[(x + a)(x + b) = x^{2} + (a + b)x + ab\]

بنابراین داریم :

\[x^{2} - 6x + 8 = (x - 2)(x - 4) = 0\]

سپس

\[x - 2 = 0\ یا\ x - 4 = 0\ \ نتیجه\ در\ x = 4\ \ یا\ x = 2\]

\[4)x^{2} - x - 6 = 0\]

\[x^{2} - x - 6 = (x - 3)(x + 2) = 0\ \ \]

\[x - 3 = 0\ یا\ x + 2 = 0\]

\[x = 2\ یا\ x = 3\]

\[5)(x + 1)^{2} + (x - 1)^{2} = x^{2} + x + 14\]

ابتدا عبارت طرف چپ معادله را ساده می کنیم :

\[(x + 1)^{2} + (x - 1)^{2} = x^{2} + x2 + 1 + x^{2} - 2x + 1 = 2x^{2} + 2\]

\[باشیم\ داشته\ باید\ پس\ :\]

\[2x^{2} + 2 = x^{2} + x + 14\]

پس از انتقال جملات به سمت چپ و ساده کردن به دست می آوریم :

\[x^{2} - x - 12 = 0\]

بنابراین :

\[x^{2} - x - 12 = (x - 4)(x + 3) = 0\]

در نتیجه ریشه های معادله عبارتند از :

\[x = 4,\ \ x = - 3\]

معادله زیر را حل کنید.

\[6)2x^{2} + 3x + 1 = 0\]

در معادلاتی که تا کنون حل کرده ایم ضریب \(x^{2}\) برابر 1 بوده است. در این مثال این طور نیست لذا معادله را به صورت زیر می نویسیم.

\[2x^{2} + 3x + 1 = \left( 2^{2}x^{2} + 3(2x) + 2 \right) = 0\]

حال قرار می دهیم \(t = 2x\) و معادله زیر را حاصل می شود :

\[t^{2} + 3t + 2 = 0\ \ \]

\[t^{2} + 3t + 2 = (t + 1)(t + 2) = 0\ \]

\[t + 1 = 0\ \ ,\ \ t + 2 = 0\ \ یا\ t = - 1\ \ ,\ t = - 2\]

بنابراین \(t = - 2\ ,\ t = - 1\) که از آن نتیجه می شود :

\[x = \frac{- 1}{2}\ \ یا\ \ x = - 1\]

4.16 حل معادله درجه دوم به روش مربع کامل کردن :

بعضی از معادلات را به روش تجزیه نمی توان حل کرد. مثلا معادله \(x^{2} - 6x + 4 = 0\) را در نظر بگیرید. پیدا کردن دو عدد که حاصل ضرب آن ها \(4\) و مجموع آنها \(- 6\) باشد ، ساده نیست (زیرا دو عدد صحیح نیستند ).برای حل این معادله آن را به صورت زیر می نویسیم.

\[x^{2} - 6x + 4 = \left( x^{2} - 6x + 9 \right) - 9 + 4 = 0\]

که با توجه به اتحاد \((a - b)^{2}\) داریم :

\[(x - 3)^{2} - 5 = 0\]

\[(x - 3)^{2} = 5\]

\[x - 3 = + \sqrt{5}\ \ \ \ یا\ \ \ \ \ x - 3 = - \sqrt{5}\]

بنابراین ریشه های معادله به صورت زیر هستند :

\[x = \sqrt{5} + 3\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ x = 3 - \sqrt{5}\]

4.17 مثال:

معادله زیر را به روش مربع کامل کردن حل کنید.

\[8)x^{2} + 4x = 12\]

معادله را به صورت زیر می نویسیم :

\[\left( x^{2} + 4x + 4 \right) - 4 = 12\]

با توجه به اتحاد \((a + b)^{2}\) داریم. \((x + 2)^{2} = 16\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \)

پس : \(x + 2 = - 4\ \ \ \ \ \ \ \ \ \ \ \ \ و\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ x + 2 = 4\)

بنابراین ریشه های معادله به این صورت هستند. \(x = 2\ \ یا\ \ \ x = - 6\)

4.18 روش کلی حل معادله درجه دوم :

برای حل معادله درجه دوم \(ax^{2} + bx + c = 0\) می توان حالتهای زیر را در نظر گرفت و در هر حالت طبق آن چه قبلا گفته شد معادله را حل کرد و ریشه ها را بدست آورد.

الف ) اگر \(c = 0\ \) باشد.آنگاه معادله به صورت \(ax^{2} + bx = 0\) در می آید که می توان آن را به صورت \(x(ax + b) = 0\) نوشت. پس ریشه های معادله عبارتند از: \(x = \frac{- b}{a},x = 0\)

4.19 مثال :

\[3x^{2} - 6x = 0 \Rightarrow 3x(x - 2) = 0 \Rightarrow 3x = 0 \Rightarrow x = 0\ ,\ x - 2 = 0 \Rightarrow x = 2\]

ب) اگر \(b = 0\) در این صورت معادله به صورت \(ax^{2} + c = 0\) در می آید.که \(x^{2} = \frac{- c}{a}\) . اگر \(- \frac{c}{a}\) مثبت باشد در نتیجه ریشه های معادله به صورت \(\pm \sqrt{\frac{- c}{a}}\) است.

4.20 مثال :

\[4x^{2} - 3 = 0 \Rightarrow 4x^{2} = 3 \Rightarrow x^{2} = \frac{3}{4} \Rightarrow x = \pm \frac{\sqrt{3}}{2}\]

ج) اگر \(b \neq 0\ \ \ \ ,\ \ \ \ c \neq 0\) در این حالت برای حل معادله درجه دوم از روش مربع کامل استفاده می کنیم. برای این منظور ابتدا \(c\) را به طرف راست می بریم. بعد دو طرف معادله را بر \(a\) تقسیم می کنیم.

  1. \(x^{2} + \frac{b}{a}x = \frac{- c}{a}\)

برای این که سمت چپ معادله مربع کامل شود به دو طرف معادله جمله \(\frac{b^{2}}{4a^{2}}\) را اضافه می کنیم ، نتیجه می شود :

3)\(x^{2} + \frac{b}{a}x + \frac{b^{2}}{4a^{2}} = \frac{b^{2}}{4a^{2}} = \frac{- c}{a}\)

  1. \(\left( x + \frac{b}{2b} \right) = \frac{b^{2} - 4ac}{4a^{2}}\)

چون سمت چپ معادله (4) همیشه نا منفی است ، لذا سمت راست این معادله نیز باید نا منفی باشد ، از طرفی \(4a^{2}\) همیشه مثبت است ، بنابراین علامت کسر سمت راست معادله (4) به علامت صورت کسر بستگی دارد. از این رو برای عبارت \(b^{2} - 4ac\) سه حالت وجود دارد.

الف ) اگر \(b^{2} - 4ac > 0\)آنگاه سمت راست معادله (4) مثبت است و معادله دارای دو ریشه است داریم.

\[x + \frac{b}{2a} = \pm \sqrt{\frac{b^{2} - 4ac}{4a^{2}}} = \pm \frac{\sqrt{b^{2} - 4ac}}{2a}\]

بنابراین ریشه های معادله (1) عبارتند از :

  1. \(x_{1} = \frac{- b}{2a} + \frac{\sqrt{b^{2} - 4ac}}{2a} = \frac{- b + \sqrt{b^{2} - 4ac}}{2a}\)

  2. \(x_{1} = \frac{- b}{2a} - \frac{\sqrt{b^{2} - 4ac}}{2a} = \frac{- b - \sqrt{b^{2}} - 4ac}{2a}\)

ب) اگر \(b^{2} - 4ac\) آنگاه معادله (4) به صورت زیر در می آید.

\[\left( x + \binom{b}{2b} \right)^{2} = \frac{0}{4a^{2}} = 0\]

که از آن \(x = \frac{- b}{2a}\) بدست می آید. همان طور که مشاهده می کنید ، برای معادله درجه دوم فقط یک ریشه بدست آمد. در این حالت معادله دارای دو ریشه مساوی یا یک ریشه مضاعف است ، یعنی :

\[x_{1},x_{2} = \frac{- b}{2a}\]

این ریشه ها را می توانیم از دستوهای (5) و (6) از حالت (الف)بدست آوریم.

پ )اگر \(b^{2} - 4ac < 0\) ،آنگاه سمت راست معادله (4) منفی بوده و معادله جواب حقیقی ندارد.

نکته : در معادله درجه دوم \(ax^{2} + bx + c = 0\) عبارت \(b^{2} - 4ac\) رامبین معادله نامیده و آن را با حروف یونانی \(\mathrm{\Delta}\)(دلتا)

نشان می دهیم.

الف ) اگر \(\mathrm{\Delta} > 0\) باشدآنگاه معادله (1) دارای دو ریشه متمایز زیر است.

\[x = \frac{- b \pm \sqrt{b^{2} - 4ac}}{2a}\]

ب ) اگر \(\mathrm{\Delta} = 0\) باشدآنگاه معادله (1) دارای یک ریشه مضاعف یا دو ریشه مساوی زیر است. \(x = \frac{- b}{2a}\)

ج) اگر \(\mathrm{\Delta} < 0\) آنگاه معادله (1) ریشه حقیقی ندارد.

مثال: معادله های زیر را حل کنید.

\[1)\ 2x^{2} - 7x + 3 = 0\]

\[2)x^{2} - 3x - 10 = 0\]

حل : الف ) مبین معادله \(2x^{2} - 7x + 3 = 0\) برابر است با :

\[\mathrm{\Delta} = b^{2} - 4ac = ( - 7)^{2} - 4(2)(3) = 49 - 24 = 25\]

چون \(\mathrm{\Delta} > 0\) بنابراین معادله دارای دو ریشه حقیقی زیر است :

\[x = \frac{- b \pm \sqrt{\mathrm{\Delta}}}{2a} = \frac{- ( - 7) \pm \sqrt{25}}{2(2)} = \frac{7 \pm 5}{4}\ \ \ \ \ یا\ \ \ x_{1} = 3\ \ ,\ \ \ x_{2} = \frac{1}{2}\]

ب) مبین معادله \(x^{2} - 3x - 10 = 0\) برابر است با :

\[\mathrm{\Delta} = b^{2} - 4ac = ( - 3)^{2} - 4(1)( - 10) = 9 + 40 = 49\]

چون \(\mathrm{\Delta} > 0\) بنابراین معادله دارای دو ریشه حقیقی زیر است :

\[x = \frac{- b \pm \sqrt{\mathrm{\Delta}}}{2a} = \frac{- ( - 3) \pm \sqrt{49}}{2(1)} = \frac{3 \pm 7}{2}\ \ یا\ \ x_{1} = 5\ \ \ \ \ \ \ ,\ \ \ \ \ x_{2} = - 2\ \]

مثال: معادله های زیر را حل کنید.

\[1)\ \ x^{2} - 6x + 9 = 0\]

\[2)2x^{2} - 7x + 7 = 0\]

حل : الف ) مبین معادله \(x^{2} - 6x + 9 = 0\) برابر است با :

\[\mathrm{\Delta} = b^{2} - 4ac = ( - 6)^{2} - 4(1)(9) = 36 - 36 = 0\]

چون \(\mathrm{\Delta} = 0\) است بنا بر این معادله دارای یک ریشه مضاعف می باشد.

\[x = \frac{- b}{2a} = \frac{- ( - 6)}{2(1)} = \frac{6}{2} = 3\]

ب) مبین معادله \(2x^{2} - 7x + 7 = 0\) برابر است با :

\[\mathrm{\Delta} = b^{2} - 4ac = ( - 7)^{2} - 4(2)(7) = 49 - 56 = 7\]

چون \(\mathrm{\Delta} < 0\) بنابراین معادله دارای ریشه حقیقی نمی باشد.

4.21 دستور محاسبه ریشه های معادله دوم در حالتی که b عددی زوج باشد :

اگر در معادله \(ax^{2} + bx + c = 0\) ، b عددی زوج باشد ،آنگاه عددی صحیح مانند \(b'\) وجود دارد که \(b = 2b'\) بنابراین در این حالت مبین معادله را می توانیم به صورت زیر بنویسیم :

\[\mathrm{\Delta} = b^{2} - 4ac = \left( 2b^{'} \right)^{2} - 4ac = 4b^{'2} - 4ac\]

اگر \(\mathrm{\Delta} \geq 0\) باشد ریشه های معادله به صورت زیر ساده می شوند :

\[x = \frac{- b \pm \sqrt{b^{2} - 4ac}}{2a} = \frac{- 2b^{'} \pm \sqrt{4(b^{'2} - ac}\ }{2a} = \frac{- 2b' \pm 2\sqrt{b^{'2} - ac}}{2a} = \frac{- b' \pm \sqrt{(b^{'2} - ac)}}{a}\]

مثال: معادله \(4x^{2} - 8x + 3 = 0\) را حل کنید.

حل : چون ضریب \((b)x\) زوج است ، ابتدا عبارت \(b^{'2} - ac\) را محاسبه می کنیم داریم

\[\left\{ \begin{matrix} a = 4\ \ \ \ \\ b^{'} = - 4 \\ c = 3\ \ \ \ \\ \end{matrix} \right.\ {\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \mathrm{\Delta}}^{'} = b^{'2} - ac = ( - 4)^{2} - 4(3) = 16 - 12 = 4\]

چون \(\mathrm{\Delta} > 0\) است پس معادله دارای دو ریشه متمایز است.

\[x = \frac{- b' \pm \sqrt{(b^{'}2 - ac)}}{a} = \frac{- ( - 4) \pm \sqrt{4}}{4} = \frac{4 \pm 2}{4}\]

بنابراین ریشه های معادله عبارتند از : \(x_{1} = \frac{3}{2}\ \ ,\ x_{2} = \frac{1}{2}\)

4.22 نکته :

اگر در معادله درجه دوم \(ax^{2} + bx + c = 0\) مجموع ضرایب برابر صفر شود \(a + b + c = 0\) در این حالت ریشه های معادله عبارتند از \(x = \frac{c}{a}\ \ ,\ x = 1\)

اثبات :

\[ax^{2} + bx + c = 0\ \ \ \ \ \ \ \ a + b + c = 0 \rightarrow a = - b - c\]

\[( - b - c)x^{2} + b + c = 0\]

\[\mathrm{\Delta} = b^{2} - 4ac = b^{2} - 4( - b - c)c = b^{2} + 4bc + 4c^{2} = (b + 2c)^{2}\]

\[x = \frac{- b \pm \sqrt{\mathrm{\Delta}}}{2a} = \frac{- b \pm \sqrt{(b + 2c)^{2}}}{2( - b - c)} = \frac{- b \pm (b + 2c)}{2( - b - c)} = \genfrac{\langle}{\rangle}{0pt}{}{\ \frac{- b + b + 2c}{2( - b - c)} = \frac{2c}{2a} = \frac{c}{a}\ \ \ \ \ \ \ \ \ \ \ }{\frac{- b - b - 2c}{- 2b - 2c} = \frac{- 2b - 2c}{- 2b - 2c} = 1}\]

ریشه های معادله \(3x^{2} + x - 4 = 0\) را بیابید.

حل : چون مجموع ضرایب \(3 + 1 - 4 = 0\) است پس ریشه های معادله عبارتند از \(x = 1.x = \frac{4}{3}\).

\[\frac{1}{2} + \frac{1}{2} - 1 = 0\]

\[x = 1,\ x = \frac{c}{a} = \frac{- 1}{\frac{1}{2}} = - 2\]

4.24 نکته :

اگر \(b = a + c\) باشد ریشه های معادله درجه دوم \(ax^{2} + bx + c = 0\) عبارتند از \(x = \frac{- c}{a}\ \ ,\ x = - 1\)

اثبات :

\[b = a + b\ \ \ ax^{2} + (a + c)x + c = 0\]

\[\mathrm{\Delta} = b^{2} - 4ac = (a + c)^{2} - 4ac = a^{2} + 2ac + c^{2} - 4ac\]

\[= a^{2} - 2ac + c^{2} = (a - c)^{2}\]

\[x = \frac{- b \pm \sqrt{\mathrm{\Delta}}}{2a} = \frac{- (a + c) \pm \sqrt{(a - c)^{2}}}{2a} = \frac{- (a + c) \pm (a - c)}{2a} = \left\{ \begin{matrix} {\ \frac{- a - c - a + c}{2a} = \frac{- 2c}{2a} = \frac{- c}{a}} \\ {\frac{- a - c - a + c}{2a} = \frac{- 2a}{2a} = - 1} \\ \end{matrix} \right.\ \]

4.25 مثال:

ریشه های معادله \(x^{2} + 3x + 2 = 0\) را بیابید.

\[\left\{ \begin{matrix} a = 1 \\ b = 3 \\ c = 2 \\ \end{matrix} \right.\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ b = 3 = a + c = 1 + 2\]

\[x = - 1\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ ,\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ x = \frac{- c}{a} = \frac{- 2}{1} = - 2\]

ریشه های معادله \(3x^{2} + 5x + 2 = 0\) را بیابید.

\[\left\{ \begin{matrix} a = 3 \\ b = 5 \\ c = 2 \\ \end{matrix} \right.\ \Rightarrow b = 5 = a + c = 3 + 2 = 5 \Rightarrow\]

\[x = - 1,\ \ x = \frac{- c}{a} = \frac{- 2}{3}\]

4.26 روابط بین ضریبها و ریشه های معادله درجه دوم

اگر معادله درجه دوم \(ax^{2} + bx + c = 0\) دارای دو ریشه متمایز یا مساوی باشد مجموع آنها مساوی با \(- \frac{b}{a}\) و حاصلضرب آنها مساوی با \(\frac{c}{a}\) است.

مجموع ریشه ها :

\[s = x_{1} + x_{2} = \frac{- b}{a}\]

\[x_{1} + x_{2} = \left\lbrack \frac{- b - \sqrt{\mathrm{\Delta}}}{2a} \right\rbrack + \left\lbrack \frac{- b + \sqrt{\mathrm{\Delta}}}{2a} \right\rbrack = \frac{- b - \sqrt{\mathrm{\Delta}} - b + \sqrt{\mathrm{\Delta}}}{2a} = \frac{- 2b}{2a} = \frac{- b}{a}\]

حاصلضرب ریشه ها :

\[p = x_{1}.x_{2} = \frac{c}{a}\]

\[x_{1}.x_{2} = \left\lbrack \frac{- b - \sqrt{\mathrm{\Delta}}}{2a} \right\rbrack\left\lbrack \frac{- b + \sqrt{\mathrm{\Delta}}}{2a} \right\rbrack = \frac{b^{2} - \mathrm{\Delta}}{4a^{2}} = \frac{b^{2} - b^{2} + 4ac}{4a^{2}} = \frac{c}{a}\]

قدر مطلق تفاضل ریشه ها :

\[D = \left| x_{1} - x_{2} \right| = \frac{\sqrt{\mathrm{\Delta}}}{|a|}\]

\[\left| x_{1} - x_{2} \right| = \left| \frac{- b + \sqrt{\mathrm{\Delta}}}{2a} - \frac{- b}{2a} \right| = \frac{\sqrt{\mathrm{\Delta}}}{|a|}\]

4.27 مثال:

بدون حل معادله \(2x^{2} - 3x - 7 = 0\) ، مجموع ، حاصلضرب و تفاضل ریشه ها را بدست آورید.

\[s = x_{1} + x_{2} = - \frac{b}{a} = \frac{2}{3}\ \ \ \ p = \ x_{1}.x_{2} = \frac{c}{a} = - \frac{7}{2}\]

\[D = \left| x_{1} - x_{2} \right| = \frac{\sqrt{\mathrm{\Delta}}}{|a|} = \frac{\sqrt{9 + 56}}{2} = \frac{\sqrt{65}}{2}\]

مثال:

\[\left\{ \begin{matrix} s = - 14 \\ p = - 51 \\ \end{matrix} \right.\ \ \Rightarrow x^{2} + 14x - 51 = 0\]

\[\Rightarrow (x + 17)(x - 3) = 0 \Rightarrow \left\{ \begin{matrix} x = - 17 \\ x = 3 \\ \end{matrix} \right.\ \]

4.35 معادله درجه دوم

\[ax^{2} + bx + c = 0\]

را که در آن \(a \neq 0\) در نظر می گیریم . این معادله را می توانیم با (( ایجاد کردن مربع کامل )) حل کنیم .

برای این منظور دو طرف معادله را در 4a ضرب می کنیم . در این صورت

\[4a^{2}x^{2} + 4abx + 4ac = 0\]

\[(2ax + b)^{2} - b^{2} + 4ac = 0\]

\[(2ax + b)^{2} = b^{2} - 4ac\]

در نتیجه اگر \(b^{2} - 4ac < 0\) ، معادله درجه دوم جواب حقیقی ندارد ، اگر \(b^{2} - 4ac = 0\) ، معدله یک جواب و اگر \(b^{2} - 4ac 0\) معادله دو جواب دارد . ( در حالتی که \(b^{2} - 4ac = 0\) ریاضیدانان می گویند که معادله همچنان دارای دو جواب است ، که این دو جواب (( برابر)) یا (( تکراری )) اند.

اگر \(b^{2} - 4ac \geq 0\) ، جوابها به صورت زیر به دست می آیند .

\[2ax + b = \pm \sqrt{{(b}^{2} - 4ac)}\]

\[x = \frac{- b + \sqrt{b^{2} - 4ac}}{2a}\]

وقتی \(b^{2} - 4ac \geq 0\) جوابها ( یا ریشه ای ) معادله \(ax^{2} + bx + c = 0\) عبارت اند از

\[a = \frac{- b - \sqrt{b^{2} - 4ac}}{2a}\]

به آسانی می توان نشان داد که رابطه زیر به ازای تمام مقادیر \(x\) برقرار است .

\[ax^{2} + bx + c = a(x - \alpha)(x - \beta)\]

این فرمول در ترسیم نمودار معادله

\[y = ax^{2} + bx + c\]

؟ است . برای این منظور نخست متذکر می شویم که وقتی \(x = \alpha\) یا \(x = \beta\) داریم داریم \(y = 0\) .

حال توجه کنید که اگر \(\alpha\) مثبت باشد آنگاه به ازای \(\beta < x < \alpha\) ، \(y\) منفی است و به ازای \(x > \alpha\) یا \(x < \beta\) ، \(y\) مثبت است . سرانجام

\[y = x^{2}\left( a + \frac{b}{x} + \frac{c}{x^{2}} \right)\]

با فرض مثبت بودن \(\alpha\) ، اگر \(x\) به قدر کافی بزرگ باشد \(y\) بزرگ و مثبت خواهد بود .

وقتی \(\alpha\) منفی است ، استدلالی مشابه استدلال فوق را می توان بکاربرد .

نمودارهای شکل 9.1 نمونه هایی از سهمی هستند . ( اگر \(b^{2} - 4ac \leq 0\) نمودارهایی مشابهی به دست می آیند ، البته اگر \(b^{2} - 4ac < 0\) این نمودار محور \(x\) ها را قطع نخواهد کرد و اگر \(b^{2} - 4ac = 0\) نمودار بر محور \(x\) ها مماس خواهد شد . )

سهمی یک از (( مقاطع مخروطی )) است . خمهای دیگر این رده مقاطع بیضی و هذلولی هستند دایره شکل 10.1 حالت خاصی از بیضی و هذلولی قائم ( وجه تسمیه قائم ، متعامد بودن مجانبهای آن است ) حالت خاصی از هذلولی است .
